Arrays and Hashes!!

There are a million enumerators out there! Sometimes it’s super hard to pick the right one for your iteration over an array or a hash. One of my personal go-to enumerators is the .group_by operation used over a hash. This operation takes a code block and returns a hash according to the information you want it to be organized by with a value. Let’s say you have an array of all of the names of your best friends because your doing a DIY project with their names, so you want to order the names by their size so you know how many letters to buy at the store. So you know you want to use the method .size iterating over each name in an array. You would do it kinda like this right? bffs =[“Kyra”, “Jenna”, “Arden”, “Megan”, “Ellen”, “Kendall”] bffs.group_by do |name| name.size Great Job! Your method will return something like this: {4=>["Kyra"], 5=>["Jenna", "Arden", "Megan", "Ellen"], 7=>["Kendall"]}
