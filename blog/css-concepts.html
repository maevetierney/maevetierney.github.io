Differences Between Margin, Border and Padding in CSS

These are three great properties in CSS! You will use them all the time. -Padding is great for separating elements and making space in content the border. Adjust in em with CSS to move the border closer or farther away from content. -The Border of the element is the edge of the element. Borders are great because they help visualize and organize an element into a confined space/box. -The Margin is the space around the element. The larger the space around it, the larger the space between other elements. Margins are helpful because thats how you move an element around the page, so its very important to know how to play with margin sizing on all 4 sides of an element to be able to space it out.
